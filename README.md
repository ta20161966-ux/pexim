# Random Task Generator

**Автор:** [Ваше Имя Фамилия]
**Описание:** Графическое приложение для генерации случайных задач с фильтрацией по типу (учёба/спорт/работа), сохранением истории в файл и возможностью добавления собственных задач.

## Возможности приложения:
*   Генерация случайной задачи из предопределённого списка.
*   Фильтрация задач по типу.
*   Отображение истории сгенерированных задач.
*   Сохранение истории в файл `history.json` (автоматически при генерации).
*   Загрузка истории при запуске приложения.
*   Добавление новых пользовательских задач с проверкой на корректность ввода.
*   История сохраняется даже после закрытия программы.

## Как запустить:
1.  Установите Python (версии 3.8+).
2.  Установите необходимые библиотеки (в данном случае только стандартные):
    `pip install -r requirements.txt` *(если потребуется что-то ещё)*
3.  Запустите файл: `python main.py`
4.  Интерфейс интуитивно понятен: кнопка для генерации, фильтр справа/сверху и поля для добавления новых задач внизу.

## Примеры использования:
1.  Запустите приложение.
2.  Нажмите кнопку «Сгенерировать задачу» — в верхней части появится случайная задача.
3.  Выберите фильтр «Спорт» — в истории будут отображаться только спортивные задачи.
4.  Введите новую задачу (например, «Помыть посуду», тип «работа») и нажмите «Добавить задачу». Теперь она будет участвовать в генерации.
5.  Закройте и откройте приложение — история будет загружена из файла.
