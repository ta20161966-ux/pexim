# Random Task Generator

**Автор:** [Ваше Имя Фамилия]
**Описание:** Приложение для генерации случайных задач с фильтрацией по типу, сохранением истории и загрузкой из JSON.
**Примеры использования:**
1. Запустить приложение.
2. Нажать «Сгенерировать задачу» — появится случайная задача.
3. Использовать фильтр для просмотра задач определённого типа.
4. Добавить новую задачу через форму (проверка на пустую строку).
5. История сохраняется в history.json и загружается при запуске.
